# ZenType

A minimalist, distraction-free typing practice tool designed for Tamil government exam aspirants.

## Why?

Most Tamil typing practice tools require complex OS keyboard configurations. ZenType solves this by building the **Marutham keyboard layout** directly into the browser.

- **Zero Setup**: Type Tamil using your standard English keyboard instantly.
- **Exam Focused**: Curated word lists relevant for government exams.
- **Pure Focus**: No ads, no distractions, just you and your wpm.
